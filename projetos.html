Projeto de Redes e Banco de Dados

Projeto Integrado de Redes e Banco de Dados, desenvolvido pelos alunos do 2º semestre de Análise e Desenvolvimento de Sistemas.

Neste projeto foi feita a criação e desenvolvimento de uma rede de computadores para produtora de música, seguindo um perfil heterogêneo com seis computadores e dois servidores atendendo os requisitos do projeto integrado de final de semestre.

Projeto de Interface com o Usuário

Projeto Integrado de Interface com o Usuário, desenvolvido pelos alunos do 3º semestre de Análise e Desenvolvimento de Sistemas.

Neste projeto foi feita a criação e desenvolvimento de um aplicativo para Campanhas e/ou Doações, integrando as disciplinas: Engenharia de Software, Orientação a Objetos e Análise e Projeto de Sistemas atendendo os requisitos do projeto integrado de final de semestre.
